# CyberOS — Interactive Cybersecurity Portfolio

[![GitHub stars](https://img.shields.io/github/stars/SolidCode321/cyberos-portfolio?style=flat-square&color=00ff88)](https://github.com/SolidCode321/cyberos-portfolio/stargazers)

**CyberOS** is a zero-dependency, ultra-fast, and highly interactive cybersecurity portfolio built to mimic a modern security-focused desktop environment (inspired by Kali Linux) and a native mobile OS (Android). It provides an immersive way to explore my skills, projects, and security research through a simulated terminal, file explorer, and more.

## 💻 Features

### 🖥️ Desktop Experience
- **Kali Linux Metaphor**: A sleek XFCE-inspired interface with a functional taskbar, start menu, and draggable windows.
- **Interactive Terminal**: A fully functional command-line interface with custom commands (`help`, `ls`, `cat`, `clear`, etc.).
- **Window Manager**: Multi-window support with minimize, maximize, and cascading capabilities.
- **Aesthetic Depth**: Glassmorphic designs, vibrant accents, and high-resolution visuals.

### 📱 Mobile Excellence (Android Mode)
- **Responsive Theme**: Automatically transitions to a portrait-optimized "Mobile OS" layout on phones and tablets.
- **Android Nav Bar**: Includes functional Back, Home, and Recents buttons for native-style navigation.
- **Mobile Status Bar**: Top bar displaying system health, connectivity, and synchronized clock.
- **Full-Screen Apps**: Windows automatically scale for the best viewing experience on small screens.

---

## 🛠️ Tech Stack
- **Languages**: HTML5, Vanilla CSS3 (Custom Properties, Grid, Flexbox), Plain ES6+ Javascript.
- **Zero Frameworks**: No React, Vue, or Tailwind. Hand-crafted for maximum performance and minimum footprint.
- **Animations**: CSS-driven hardware-accelerated animations for smooth transitions.
- **Icons**: Centralized library of custom, scalable SVGs.

## 🕵️ Capture The Flag (CTF)
CyberOS isn't just a portfolio; it's a playground. There are hidden secrets throughout the system that only a true security enthusiast will find.

> [!TIP]
> **"Not everything that is hidden is lost..."**
> If you find yourself in the terminal, remember that a detailed investigation (`-la`) of the environment might reveal what the surface layer conceals. Sometimes, the most important files are the ones that start with a dot.

## 🚀 Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/SolidCode321/cyberos-portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd cyberos-portfolio
   ```
3. Open `index.html` in your browser (or use a Live Server extension).
